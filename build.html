Creating New Test Accounts

  1. Go to https://rentmycar-demo.netlify.app
  2. Click "Sign In" button
  3. Click "Sign Up" link at the bottom
  4. Fill in your details and select Role (Renter or Owner)
  5. Click "Create Account"

Feature Testing Checklist

Homepage Features

  • Car listings display with images, prices, and ratings
  • Filter by car type (Sedan, SUV, Sports, etc.)
  • Price range slider filtering
  • Distance filter
  • Book Now buttons functional
  • Sign In / Sign Up modals
  • Responsive design for mobile

Owner Dashboard

  • Overview: Stats cards (Earnings, Rentals, Bookings, Rating)
  • Fleet Manager: Add new vehicle with full registration
  • Fleet Manager: Edit existing vehicles
  • Fleet Manager: Delete vehicles
  • Bookings: View pending/approved/completed bookings
  • Bookings: Approve/Decline booking requests
  • Earnings: Transaction history
  • Analytics: Charts and statistics
  • Maintenance: Schedule vehicle maintenance
  • Reviews: View customer reviews
  • Settings: Profile, notifications, payment settings

Vehicle Registration Requirements

  • Vehicle Details: Make, Model, Year, Type, Color
  • VIN (17-character validation)
  • Registration: License Plate, State, Expiry Date
  • Insurance: Provider, Policy Number, Expiry Date
  • Document Uploads: Driver's License, Registration, Insurance Card
  • Vehicle Photos upload
  • Terms & verification confirmation

Renter Features

  • Browse available cars
  • Filter and search functionality
  • Book vehicles
  • View booking history
  • Track active rentals

Technology Stack

Component Technology

Backend API Google Apps Script

Icons Font Awesome 6

Browser Compatibility

  • Google Chrome (Recommended)
  • Microsoft Edge
  • Mozilla Firefox
  • Safari

Known Limitations (Demo Environment)

  • Document uploads store file names only (production would use cloud storage like AWS S3)
  • Stripe integration is in test mode
  • Some dashboard data is simulated for demonstration
  • Privacy-focused browsers may require allowing cookies/storage

API Endpoints

Endpoint Method Description
/auth/signup POST Create new account
/auth/login POST User login
/auth/logout POST User logout
/cars GET List all cars
/cars POST Create car listing
/cars/:id PUT Update car
/cars/:id DELETE Delete car
/bookings GET/POST List/Create bookings
